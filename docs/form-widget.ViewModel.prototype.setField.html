setField

  • function
 

Sets the formObject value when a field changes. This will allow for future functionality where the form is much more responsive to values changing, like cascading dropdowns.

function(field, domElement, event, value)

Parameters

  1. field {Field}

    The field object properties

  2. domElement {domElement}

    The form element that dispatched the event

  3. event {Event}

    The event object and type

  4. value {Object | Number | String}

    The value that was passed from the field component
