addFilter

  • function
 

Adds a new filter or set of filters to the list of filters in this widget. A filterFactory may be defined on the field which may return one filter or an array of filters.

function(scope, dom, event, name)

Parameters

  1. scope {can.Map}

    The stache scope (optional)

  2. dom {event}

    The dom event (optional)

  3. event {event}

    The event (optional)

  4. name {string}

    The name of the field filter to add. This is the only argument used by the function, the rest may be null.

Returns

{Boolean}

returns false to prevent event propagation from links
