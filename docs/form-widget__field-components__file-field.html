<file-field />

  • module
form-widget/field-components/file-field

{can.Component}

 
  • Input Accept Types

A file upload field. Once a file is selected, this component POST's the upload to a special url and returns the url result as text to the form.

can.Component

Properties

  1. fieldType {String}

    Should be set to file for the form to use this template

  2. url {String}

    The url up post the file upload to

  3. multiple {Boolean}

    Whether or not to allow multiple uploads

  4. accept {String}

    The type of media to accept
