list-table.TableButtonObject

{TableButtonObject}

 

Used by the list-table component to provide custom interactivity with each object or row.

TableButtonObject

Properties

  1. title {String}

    The title to display on the button hover

  2. iconClass {String}

    The class to use for the button icon

  3. eventName {String}

    The event to dispatch when the button is clicked. This allows developers to bind functions to custom events, like (eventName)="functionName"
